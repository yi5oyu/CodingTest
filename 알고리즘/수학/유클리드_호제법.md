## 최대공약수(GCD, Greatest Common Divisor)

`시간 복잡도: O(log min(a, b))`  
`GCD(a, b) = GCD(b, a % b)`   

`q: a/b`  
`r: a%b`  
`a = b * q + r`  
`a,b 공약수 = b, r 공약수`
`나머지 b가 0일 때까지 반복`


### 구현

```
// 반복문
public static int gcd(int a, int b) {
      while (b != 0) {
          int temp = b;
          b = a % b;
          a = temp;
      }
      return a;
  }

// 재귀
public static int gcd(int a, int b) {
    if (b == 0) {
        return a;
    }
    return gcd(b, a % b);
}
```

## 최소공배수(LCM)

`LCM(a, b) = (a × b) / GCD(a, b)`
