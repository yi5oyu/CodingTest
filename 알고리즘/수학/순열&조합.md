# **순열 & 조합**

순열(Permutation): 순서가 중요한 경우의 수  
조합(Combination): 순서가 중요하지 않은 경우의 수

`완전 탐색`

</div>

## 순열

`서로 다른 n개 중에서 r개를 순서대로 나열하는 경우의 수`  
`nPr = n! / (n-r)!`  

### 구현

`백트래킹`

```
int n, r;
int[] arr;


```

## 조합

`서로 다른 n개 중에서 r개를 선택하는 경우의 수`  
`nCr = n! / (r! × (n-r)!)`
