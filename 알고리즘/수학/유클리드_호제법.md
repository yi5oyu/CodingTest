# **유클리드 호제법**

## 최대공약수(GCD, Greatest Common Divisor)

`시간 복잡도: O(log min(a, b))`  
`GCD(a, b) = GCD(b, a % b)`   

`q(몫): a/b`  
`r(나머지): a%b`  
`a = b * q + r`  
`a,b 공약수 = b, r 공약수`
`나머지 b가 0일 때까지 반복`

```java
// GCD(48, 18)
48 = 18 × 2 + 12  // GCD(48, 18) = GCD(18, 12)
18 = 12 × 1 + 6   // GCD(18, 12) = GCD(12, 6)
12 = 6 × 2 + 0    // GCD(12, 6) = 6
```

### 구현

```
// 반복문
int a = 15;
int b = 3;

while (b != 0) {
    int temp = b;
    b = a % b;
    a = temp;
}
// a = 3(최대공약수)

// 재귀
public int gcd(int a, int b) {
    if (b == 0) {
        return a;
    }
    return gcd(b, a % b);
}
```

## 최소공배수(LCM, Least Common Multiple)

`LCM(a, b) = (a × b) / GCD(a, b)`

## 3개 이상의 수

```java
int[] arr = {24, 36, 48};
int result = arr[0];

for (int i = 1; i < arr.length; i++) {
    int a = result;
    int b = arr[i];
    
    // 최대공약수 계산
    while (b != 0) {
        int temp = b;
        b = a % b;
        a = temp;
    }
    result = a;
    
    if (result == 1) break;  // 1이면 더 계산 필요없음
}
```
